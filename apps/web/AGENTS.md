# Open Science Web Dashboard

This is a Next.js application serving as the public dashboard for the Open Science distributed research platform.

## Development
- `bun dev` to start the development server
- Uses App Router with server components by default
- Tailwind CSS for styling
- Will connect to Convex for real-time data

## Structure
- `src/app/` - App Router pages and layouts
- `src/components/` - Reusable UI components
- `src/lib/` - Utilities and Convex client setup
